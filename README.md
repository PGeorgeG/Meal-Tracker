# Mise en Bouche

AI-powered meal journal. Tracks calories, protein, carbs, and fat with Claude estimates.

## Deploy to Railway

1. Push this folder to a GitHub repo
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Select your repo — Railway auto-detects the config and deploys
4. Your app will be live at the Railway-provided URL in ~1 minute

## Local development

```bash
npm install
npm start
# Open http://localhost:3000
```

## Usage

On first load, enter your Anthropic API key (starts with `sk-`).  
It's stored in your browser's localStorage — never on the server.
